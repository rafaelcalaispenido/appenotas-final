Detalhe do erro de emissão
NFS-E · CÓDIGO E160
Rejeição ao emitir NFS-e — Código E160: Arquivo em desacordo com o XML Schema
O que significa este erro?

O código E160 indica que o arquivo XML enviado para a prefeitura está fora do padrão esperado pelo XML Schema do município. Isso ocorre quando há campos obrigatórios ausentes, tipos de dados incorretos ou formatação inválida no documento fiscal.

Mensagem retornada pela prefeitura
E160 – Arquivo em desacordo com o XML Schema. Verifique os campos obrigatórios e os tipos de dados informados.
Possíveis causas
  • Campo obrigatório não preenchido (ex.: CNPJ do prestador, código do serviço LC116)
  • Formato de data ou valor numérico incorreto
  • Caracteres especiais não escapados no XML (&, <, >)
  • Versão do schema desatualizada para o município
  • Tag XML duplicada ou fora da hierarquia esperada
Como resolver

Verifique as configurações da empresa, especialmente o código do serviço (CNAE / LC 116) e os dados do prestador e tomador. Se o problema persistir, acesse o artigo completo na Central de Ajuda ou entre em contato com o suporte informando o número da venda e a mensagem de erro.
